**Uptime Pro** is a high-performance, aesthetically stunning web analytics tool designed to track your browser session duration with precision. Built with a sleek glassmorphic interface, it offers real-time session monitoring and detailed 7-day retrospective analytics.

![Uptime Pro Hero](favicon.png)

## ✨ Key Features

- ⏱️ **Zero-Drift Tracking**: Uses high-precision timestamp delta calculation for millisecond-perfect uptime tracking.
- 📊 **Dynamic Analytics**: Interactive bar charts visualizing your last 7 days of activity in hours.
- 🎨 **Premium UI/UX**: Modern glassmorphic design featuring glowing neon accents and responsive layouts.
- 💾 **Persistent Session**: Automatic local vault backup every 3 seconds to ensure no data is lost during breaks or crashes.
- ⏯️ **Session Control**: Seamlessly pause and resume tracking whenever you need a breather.

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Raza023/browseruptime.git
   ```
2. **Open the project:**
   Simply open `index.html` in your favorite browser.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design System)
- **Logic**: Vanilla JavaScript (ES6+)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)
